== Welcome to Transit & Trails (on Rails)

Transit & Trails is platform for finding, planning and sharing outdoor activities.

The platform consists of :

* A data API for managing trips, trailheads, campgrounds
and related objects.

* A consumer web application for accessing those objects.

* iPhone and Android mobile applications for accessing those objects.

* Embeddable widgets for planning and creating new trips

Coming soon :

* Responsive mobile optimized web application
